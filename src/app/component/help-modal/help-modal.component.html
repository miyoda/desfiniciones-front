Ayuda Como conectar
Desfiniciones es un juego para jugar en familia, con los amigos o con enemigos...pero no para jugar solo.
Empezar a jugar es fácil y rápido:
Que uno del grupo cree una sala y proporcione el código de esta sala al reso para que se unan a la partida.
 Como jugar
1
El sistema seleccionará aleatoriamente una palabra real pero usualmente poco conocida.
2
Cada jugador escribirá una "desfinición" para esa palabra.
3
Se mostrarán todas las "desfiniciones" con la definición real y se votará.
4
El objetivo es acertar cual es la definición verdadera... y conseguir engañar al resto para que voten la tuya es la verdadera.
